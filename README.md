# 📦 Supply Chain & Distribution Analytics System

## 🔍 Overview

This project is an end-to-end Supply Chain & Distribution Analytics System built using Excel to simulate real-world operations across inventory, logistics, suppliers, and distribution networks. It integrates multiple operational components into a single system to improve visibility, efficiency, and decision-making.

## 🎯 Objective

* Analyze inventory, logistics, and distribution performance
* Identify bottlenecks in supply chain operations
* Improve demand planning and inventory management
* Track key operational KPIs
* Enable data-driven decision making

## 📊 Project Features

* Inventory Analysis (ABC Classification, Turnover, Stock Levels)
* Demand Forecasting (Moving Average, Trend Analysis)
* Supplier Performance Evaluation (Lead Time, Reliability)
* Logistics & Distribution Analysis (On-Time Delivery, Delays)
* Cost Analysis (Freight, Procurement, Holding Costs)
* KPI Dashboard (Inventory Turnover, Fill Rate, OTD %, Cost Metrics)

---

## 📈 Key Insights

* Identified high-value inventory segments requiring tighter control
* Detected regional delivery delays impacting service levels
* Highlighted cost drivers in logistics and distribution operations
* Improved visibility into supplier performance and reliability
